## Common Techniques

- **Two Pointers**: Used in problems like two_sum, three_sum, trap
- **Sliding Window**: Used in substring problems like length_of_longest_substring
- **Hash Maps**: Used for lookup optimization in problems like two_sum, group_anagrams
- **In-place Operations**: Used in problems like move_zeroes, rotate
- **String Manipulation**: Used in problems like is_palindrome, str_str
